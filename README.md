# postmarket_samsung_a52q
postmarketOS for Samsung Galaxy A52

## Works
- USB Gadget mode
- Display (simplefb)
- CPU (tlmm, clock controllers)
- UFS Storage
- SDHC (SDCard)
- IOMMU
- Battery (sm5714-fg)
- Some other things from sc7180.dtsi
## Not tested
- Display (samsung,s6e3fc3-ams667ym01)
- GPU (FD618)
## Broken
- Display Brightness
- Modem
- Wifi
- Bluetooth
- Touchscreen
## Bugs
- Modem sometimes can't start at boot.
